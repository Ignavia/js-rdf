    public parse(s: String, options: Object): Promise source

    Parses a Turtle string an returns a graph and a profile.

    Params:

    NameTypeAttributeDescription
    s String

    The Turtle string to parse.
    options Object

    The options object.
    options.filter Function
    • optional

    Determines which triple to include in the resulting graph.
    options.graph Graph
    • optional

    The graph to add the triples to.
    options.profile Profile
    • optional

    The profile to add the prefixes and terms to.

    Return:

    Promise

    A promise that eventually resolves to the resulting graph and profile. If there is an error, the promise is rejected.
