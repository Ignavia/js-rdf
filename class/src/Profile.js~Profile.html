    public importProfile(profile: Profile, override: Boolean): Profile source

    This method adds all prefixes and terms from the given profile. It allows easy updating and merging of different profiles, such as those exposed by parsers.

    Params:

    NameTypeAttributeDescription
    profile Profile

    The profile to import.
    override Boolean

    If true then conflicting terms and prefixes will be overridden by those specified on the Profile being imported. By default imported terms and prefixes augment the existing set.

    Return:

    Profile

    This profile to allow chaining.

    See:

    • https://www.w3.org/TR/rdf-interfaces/#widl-Profile-importProfile-Profile-Profile-profile-boolean-override

    public nodeToString(node: RDFNode, conf: Object): String source

    Returns a string representing the given RDFNode. Literals might be replaced with terms stored in this profile, while names nodes might be shrunk to a CURIE. If no matches are found in this profile, the normal toString method is called.

    Params:

    NameTypeAttributeDescription
    node RDFNode

    The node to stringify.
    conf Object
    • optional

    The configuration object.
    conf.shrinkTerms Boolean
    • optional
    • default: true

    Whether terms should be shrunk.
    conf.shrinkPrefixes Boolean
    • optional
    • default: true

    Whether prefixes should be shrunk.

    Return:

    String

    The resulting string.
